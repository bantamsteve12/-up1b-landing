UP/1B Calculator is a mobile application designed to guide Gas Safe registered engineers through the tightness testing and purging procedure set out in IGEM/UP/1B Edition 4. It provides step-by-step guidance, installation volume calculations, pressure drop assessments, and exportable test records.

The App is available on iOS via the Apple App Store and on Android via Google Play.
